🎬 Animation Pipeline – Vorschau

SVG- und Lottie-Animationen im Browser ansehen. Datei wählen, hierher ziehen oder Pfad laden.

Noch nichts geladen.
Wähle eine .svg- oder .json-Datei (Lottie), ziehe sie hierher, oder lade einen Pfad.

Tipp: Für „Pfad laden" das Repo per python -m http.server ausliefern und http://localhost:8000/preview/ öffnen (direkter Datei-Aufruf blockiert das Laden via Browser-Sicherheitsrichtlinien).
